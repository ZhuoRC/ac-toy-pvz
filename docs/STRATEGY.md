# Game Strategy Guide

## Early Game (Waves 1-2)
1. Build 3-4 Sunflowers first
2. Add Peashooters row by row
3. Don't overspend

## Mid Game (Waves 3-4)
1. Upgrade to Repeaters
2. Add Wall-nuts at front
3. Focus fire on Football Zombies

## Late Game (Wave 5)
1. All Repeaters
2. Double Wall-nut layers
3. Survive the final wave!
